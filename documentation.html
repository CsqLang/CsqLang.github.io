Introduction

Csq is a high level easy and transpiled programming language whose syntax is very much similar to Python and having the speed of C++.

It was designed by Aniket Kumar in 2022.

Getting Started

There are total two functional versions of Csq but in this case we are going to use version (4.2)

                    
                      $ git clone https://github.com/CsqLang/Csq4 
                      $ cd Csq4/
                      $ bash build.sh
Now you are good to go!!

Hello World program

Let's create our first program in Csq

  • Create a file hello.csq
  • Write code inside it:
    •                 
                  print('Hello, World!')
  • Let's run it
    •                 
                  $ ./csq cpp hello 'current-directory' 
                  
    
                >> Hello, World!

    Data Types

    Csq is a statically typed programming language so the data types of a variable can't be changed

    Builtin data-types
  • int
  • float
  • double
  • list
  • str

    • By default Csq uses type inference so no type checking will be there during transpilation but at last it will be there.

    Creating a Variable

                    
                  a = 405
                  print(a)
                
                
    Output:
    
                >> 405

    By default variables use stack memory which increases the performance.

    Control Structures

    Csq supports all types of control structures present in Python.

    But be aware of the indentation since Csq doesn't consider tabs as indents.

    If stmt

    If statement will exectue a block of statement only when condition is true.

                    
                  a = 334
                  b= 334
                  if a == b:
                   print("equal")
                
                
    Output:
    
                >> equal

    Else stmt

    else statement will exectue a block of statement only when if condition is false.

                    
                  a = 334
                  b = 3334
                  if a == b:
                   print("equal")
                  else:
                   print("not equal")
                
                
    Output:
    
                >> not equal

    Elif stmt

    elif statement will exectue a block of statement only when above if or another elif condition is false.

                    
                  a = 334
                  b = 3334
                  if a == b:
                   print("equal")
                  elif a < b:
                   print('done')
                
                
    Output:
    
                >> done

    Loops

    To repeat a piece of code we use loop, In Csq there are two types of loop while and for.

    For loop

    It's a type of loop which iterates the operand given.

                    
                    for i in range(5):
                     print(i)
                
                
    Output:
    
                >> 
                0
                1
                2
                3
                4

    While loop

    It's a type of loop which exectues the block of code until condition is false.

                    
                    i = 0
                    while i<5:
                     print(i)
                     i = i+1
                
                
    Output:
    
                >> 
                0
                1
                2
                3
                4

    Functions

    They are the piece of code which can be called and used whenever we want.

                    
                    def sum(a = 0,b = 0):
                     return a+b
                    print(sum(2,4))
                
                
    Output:
    
                >> 6
    Note: functions in Csq doesn't support recursion since it's causes stack overflow and makes the program slow.
